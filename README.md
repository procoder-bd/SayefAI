# SayefAI

A personal AI desktop assistant built with Python.

## Features

- AI Chat
- Windows Automation
- Browser Automation
- Voice Assistant
- Memory
- Plugin System
- GoHighLevel Automation
- Developer Tools

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/SayefAI.git

cd SayefAI

python -m venv .venv

.venv\Scripts\activate

pip install -r requirements.txt

playwright install

python main.py
```
